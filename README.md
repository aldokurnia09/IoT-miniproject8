# IoT-Mini Project 8
ESP8266 dihubungkan dengan Firebase, kemudian akan menampilkan value dari komponen tersebut ke dalam aplikasi android
--------------------------------------------------

### Hardware yang digunakan: 
- NodeMCU ESP8266
- 1 LED
- Sound Sensor
- Water Level Sensor
- Kabel Jumper
- Project Board

### Software yang digunakan: 
- Arduino IDE
- Android Studio
